Online Course Evaluation:

Overall, the courses offered by lab.github.com provided a great working knowledge of Github. The interactive nature of the courses made sure that the content was not just delivered in the abstract. While there were sometimes problems with the bots not sending you to the next step in the course, the team at tech support at Github was extremely responsive to posts in the community lab forums. The only flaw in the delivery of this content was that I waited until the end of the semester before I began creating my Github page for my final project. After months of not using this tool, I needed to retake the courses to make sure I could complete the assignment. I would suggest tasking students with the creation of their Github project template page within the first few weeks of the semester.
