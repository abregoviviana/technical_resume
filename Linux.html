Key Concepts:

Before starting the online course for Linux you should install software for running virtual machine on your local PC. I used Oracle VM VirtualBox (which is used in the install demo as well). You should also go ahead and download the Ubuntu .iso file. While the videos walk through this process, it is a lot easier to follow along with the files are ready to go before you begin.

When going through the demonstrations that within this course, the major issue I continuously ran into was case sensitivity. While I think the videos do an okay job of explaining why Linux is case sensitive, it is still extremely hard to get used to and can consistently mess you up.

Working in the VI editor was the most challenging part of the play along demonstrations. There is not a good way to tell if you are in command or input mode, so it is very easy to mess up your script and hard to fix. My suggestion would be to use something VIM or Nano which is a little more user friendly.

Online Course Evaluation:

I believe the Guru99 Introduction to Linux courses provided a great beginners guide to the Ubuntu Distro of a Linux desktop. While this content does not have labs or quizzes like the Github or edX materials, the videos are written in such a way that the user can follow along with all the steps on their own Ubuntu console. My major recommendation would be to require this course earlier in the semester (before the students take PowerShell), as I believe it would provide a better base foundation of OS management and scripting.
